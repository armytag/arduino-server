# Arduino Server

> A full webserver hosted on an Arduino Uno

## Requirements
#### Hardware
- Arduino Uno
- Ethernet Shield
- microSD card

#### Software
- Arduino IDE
- Arduino Ethernet library
- Arduino SD library

## Installation

Just download the arduino file and upload it to your Arduino!
